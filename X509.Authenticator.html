Module X509.Authenticator

Authenticators of certificate chains

Authenticators


type a = ?host:X509.host -> X509.t list -> X509.Validation.result
An authenticator a is a function taking a hostname and a certificate stack to an authentication decision X509.Validation.result.
val chain_of_trust : ?time:float -> X509.t list -> a
chain_of_trust ?time trust_anchors is authenticator, which uses the given time and list of trust_anchors to verify the certificate chain. This is an implementation of the algorithm described in RFC5280, using X509.Validation.verify_chain_of_trust.
val server_fingerprint : ?time:float ->
       hash:Nocrypto.Hash.hash ->
       fingerprints:(string * Cstruct.t) list -> a
server_fingerprint ?time hash fingerprints is an authenticator which uses the given time and list of fingerprints to verify the first element of the certificate chain, using X509.Validation.trust_fingerprint.
val null : a
null is authenticator, which always returns `Ok. For testing purposes only.
val a_of_sexp : Sexplib.Sexp.t -> a
a_of_sexp sexp is authenticator, the unmarshalled sexp
val sexp_of_a : a -> Sexplib.Sexp.t
sexp_of_a authenticator is sexp, the marshalled authenticator
